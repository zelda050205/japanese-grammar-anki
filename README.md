# 日语语法记忆系统 - Anki风格

一个基于Anki算法的日语语法学习系统，支持智能间隔重复、语法高亮、多课时上传等功能。

## 🎯 功能特点

### 📚 核心功能
- **26课完整框架**：第01课～第26课日中对照标题
- **Anki算法**：严格按照Anki底层逻辑实现间隔重复
- **智能语法高亮**：例句中的语法点自动标记为蓝色
- **多课时上传**：支持一个文档包含多个课程
- **学习进度跟踪**：完整的统计和进度管理

### 🔍 语法高亮特点
- 智能识别课程中所有语法点
- 例句中的语法部分显示为蓝色
- 假名标注显示在汉字上方
- 精确匹配避免重复标记

### 📱 移动端优化
- 响应式设计，完美适配手机屏幕
- 触摸友好的操作界面
- 流畅的动画和过渡效果

## 🚀 部署到Gitee Pages

### 步骤1：创建Gitee仓库
1. 登录 [Gitee](https://gitee.com/)
2. 点击右上角"+"号，选择"新建仓库"
3. 填写仓库名称（如：japanese-grammar-anki）
4. 设置为公开仓库
5. 点击"创建"

### 步骤2：上传文件
1. 在仓库页面点击"上传文件"
2. 将以下文件上传到仓库：
   ```
   ├── index.html          # 主页面
   ├── styles/
   │   └── main.css        # 样式文件
   ├── js/
   │   ├── app.js          # 主应用逻辑
   │   ├── data.js         # 课程数据
   │   ├── storage.js      # 存储管理
   │   └── spaced-repetition.js  # 间隔重复算法
   └── README.md           # 说明文档
   ```

### 步骤3：启用Gitee Pages
1. 在仓库页面找到"服务"菜单
2. 点击"Gitee Pages"
3. 选择"master"分支
4. 部署目录选择"/"（根目录）
5. 点击"启动"

### 步骤4：访问网站
- 部署成功后，会生成一个访问链接
- 格式：`https://用户名.gitee.io/仓库名`
- 在手机浏览器中打开此链接即可使用

## 📖 使用说明

### 上传语法内容
支持的txt文档格式：
```
第2课
忘れないうちに、カレンダーにメモしておこう。
趁着没忘，先记在日历上吧。
「〜うちに」：A:"趁着～赶紧"+意志；B:后+不知不觉、自然而然的变化；不+意志

第3课
彼女は毎日遅刻ばかりしている。
她每天都在迟到。
「〜ばかりだ」：表示不断重复某种不好的行为
```

### 学习流程
1. **上传语法**：点击"📤 上传语法"上传txt文档
2. **开始学习**：点击"📚 学习新语法"选择课程
3. **复习巩固**：点击"🔄 复习语法"复习到期卡片
4. **查看进度**：点击"📊 学习统计"查看学习数据

### Anki算法说明
- **重来**：1分钟后重新出现
- **困难**：6分钟后重新出现
- **良好**：根据学习阶段调整间隔
- **简单**：更长的复习间隔

## 🔧 自定义配置

### 修改课程标题
编辑 `js/data.js` 文件中的 `lessonsData` 对象

### 调整学习参数
编辑 `js/spaced-repetition.js` 文件中的算法参数

### 样式定制
编辑 `styles/main.css` 文件自定义界面样式
